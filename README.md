# module-leyu-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件的形式存放，不依赖特定域名或后端服务，可直接通过浏览器打开或部署到任何静态托管平台。

仓库本身不限定页面的主题或用途，主要作为 HTML 页面集合的版本管理与发布中心。

## 目录结构

```
module-leyu-guide/
├── pages/              # 存放所有独立 HTML 页面文件
├── assets/             # 页面引用的资源文件（如样式、图片、脚本等）
├── README.md           # 本文件
└── LICENSE             # 项目许可证
```

- `pages/`：每个 HTML 文件为一个独立页面，文件名即页面标识。
- `assets/`：按需组织子目录，用于存放页面所需的静态资源。

## 页面归档说明

- 所有页面均为纯静态 HTML，无服务器端依赖。
- 页面之间相互独立，不共享状态或数据。
- 归档内容可能随时间更新或增删，请以仓库最新提交为准。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接使用浏览器打开 `pages/` 目录下的 HTML 文件即可查看。
3. 也可将整个仓库部署到任何支持静态文件的 Web 服务器或托管服务。

## 维护说明

- 本仓库由维护者不定期更新。
- 如需新增页面，请遵循现有目录结构，将 HTML 文件放入 `pages/`，相关资源放入 `assets/`。
- 提交更新时，请尽量附带简要的提交说明，以便追踪变更。

## 许可

本仓库内容遵循仓库内 LICENSE 文件中的许可条款。请在使用前查阅。
